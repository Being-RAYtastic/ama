# Ask Me Anything 💬

A simple AMA (Ask Me Anything) web app where anyone can submit questions, and replies are managed through Firebase Firestore.

## Features

- Submit questions via a clean textarea input
- Questions stored and fetched from Firebase Firestore
- Replies displayed below each question
- Auto-sorted by newest first
- Scrollable question list
- Fully responsive UI


## Setup Locally

1. Clone the repo.

2. Install dependencies:
```bash
npm install
```
3. Create `.env` in the project root with your Firebase config values.

4. Run locally:
```bash
npm run dev
```
